## Installation

You can add this library to your Rust project directly from crates.io or Git repository.

## crates.io

```toml
the-compiler-strikes-back = "0.1.0"
```

## Git repository

Add **one** of the following options under `[dependencies]` in your `Cargo.toml`.

### Using SSH

```toml
the-compiler-strikes-back = { git = "ssh://git@github.com/TheCompilerStrikesBackAP2025/TheCompilerStrikesBack.git"}
```
### Using HTTPS

```toml
the-compiler-strikes-back = { git = "https://github.com/TheCompilerStrikesBackAP2025/TheCompilerStrikesBack.git"}
```
