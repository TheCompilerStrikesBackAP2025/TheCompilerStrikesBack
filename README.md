## Installation

You can add this library to your Rust project directly from the Git repository.

Add **one** of the following options under `[dependencies]` in your `Cargo.toml`.

### Using SSH

```toml
TheCompilerStrikesBack = { git = "ssh://git@github.com/TheCompilerStrikesBackAP2025/TheCompilerStrikesBack.git"}
```
### Using HTTPS

```toml
TheCompilerStrikesBack = { git = "https://github.com/TheCompilerStrikesBackAP2025/TheCompilerStrikesBack.git"}
```
